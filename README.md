# Canopy

**Budget & Carbon-Aware Infrastructure Architect**

An open-source framework for unified cloud cost and carbon optimization, powered by agentic AI.

## Quick Start

```bash
pip install canopy-cloud

# Show region efficiency tiers
canopy regions

# Audit your AWS infrastructure
canopy audit --provider aws

# Preview cost/carbon impact of IaC changes (coming in v0.2)
canopy plan
```
